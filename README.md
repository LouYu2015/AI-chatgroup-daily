# AI 群聊日报

每日 AI 技术讨论精选，经过匿名化处理后公开发布。

🌐 **站点地址**：[louyu2015.github.io/AI-chatgroup-daily](https://louyu2015.github.io/AI-chatgroup-daily)

---

## 关于

本站发布一个关注 AI 技术的微信群的每日讨论精选。每篇日报由 AI（Claude Opus）从当天的群聊记录中提取整理，保留有价值的技术讨论、工具分享与方法论交流。

## 隐私保护

群友发言经过三级隐私处理：

- **Optout（不出现）**：发送 `/optout` 后，该用户的所有发言从公开版完全移除
- **默认匿名**：所有群友默认以稳定的匿名名（如「沉稳的大象07」）出现
- **公开别名**：通过 `/alias <名字>` 指令可设置个性化别名

## 群内指令

在群内发送以下指令（需单独成行，不会实时回复，在下一份日报中生效）：

| 指令 | 说明 |
|------|------|
| `/alias <名字>` | 设置公开别名（1–16 字符） |
| `/alias` | 清空别名，恢复默认匿名 |
| `/optout` | 退出公开版 |
| `/optin` | 重新参与公开版 |
